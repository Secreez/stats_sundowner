1. Lade Git-Repository herunter

2. installiere mind: 
    - Python 3.x
    - pip
    - streamlit
    - pandas
    - bokeh

3. starte streamlit mit: 
    - streamlit run main.py

4. öffne Browser mit: 
    - http://localhost:8501

5. Solltest du beenden wollen:
    - drücke `Strg + C` im Terminal!

6. Viel Spaß beim Ausprobieren! =)
